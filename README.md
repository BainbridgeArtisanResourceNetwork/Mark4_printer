# Mark4_printer
Mark4 is the large-format 3D printer the ETA studio at BARN is building as a DIY community project.

## Specifications

- Motion stage: CoreXY
- Build volume: X=350mm, Y=350mm, Z>=400mm
- Controller: Duet 2 WiFi
- Extruder/Hot-end: Bondtech BMG-M & Mosquito
